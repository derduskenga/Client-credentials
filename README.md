# Client-credentials
## How to use: 
1. clone the repo
1. Rename the `.env.example` file to `.env`
1. In the `.env` file, add values for the following variable:
    - SERVER_PORT
    - APP_CLIENT_ID
    - APP_CLIENT_SECRET
    - SUSI_POLICY_AUTHORITY
    - AUTHORITY_DOMAIN
1. From your terminal, run `npm install`
1. From your terminal, run `node .index.js`
